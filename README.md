# Network Traffic Analysis (Wireshark)

## Overview

This project demonstrates practical network traffic analysis using Wireshark and OSI model concepts.

## Topics Covered

* DNS resolution
* TCP three-way handshake
* TLS handshake
* Packet loss and retransmissions
* Duplicate ACK analysis

## Tools Used

* Wireshark
* Windows OS

## Key Skills

* Network troubleshooting
* Packet analysis
* Understanding of OSI model
